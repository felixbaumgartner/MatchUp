# MatchUp

A browser-based memory card game built with vanilla HTML, CSS, and JavaScript. Flip cards to find matching emoji pairs in the fewest moves and shortest time.

## How to Play

1. Click any card to flip it face-up
2. Click a second card to find its match
3. If the cards match, they stay revealed. If not, they flip back after 1 second
4. Match all 8 pairs to win!

## Features

- 4x4 grid with 8 randomly shuffled emoji pairs
- Smooth 3D card flip animations
- Move counter and timer
- Win screen with final stats
- New Game / Play Again buttons
- Responsive design (desktop and mobile)

## Getting Started

Simply open `index.html` in any modern browser — no build tools or dependencies required.

To run on localhost:

```bash
npx serve
```

## Tech Stack

- **HTML** — game layout and structure
- **CSS** — card styling, flip animations, responsive grid
- **JavaScript** — game logic, state management, DOM manipulation

No frameworks or libraries. Pure vanilla web.
